====================================================================
This directory contains a paper with benchmarks for Curry systems,
in particular, KiCS2:
====================================================================

Generate and show the paper by

> ../execpaper -f bench_kics2.tex

Notes:

* the benchmarks are defined for a specific environment but can be
  very easily adapted
  (see the definitions in the header of BenchKiCS2, e.g., kics2Systems,
  binOfSystem)

* the execution takes *very* long time
